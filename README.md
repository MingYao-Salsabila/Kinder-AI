# KinderAi

KinderAi is a modular AI learning assistant built for the **MajuBarengAi programme by Hacktiv8 with Google**.

## Modes
- Kid Mode
- Teacher Mode
- Admin Mode

## Quick start
1. Create a Python virtual environment.
2. Install dependencies from `requirements.txt`.
3. Configure environment variables from `.env.example`.
4. Run `streamlit run app/main.py`.

## Scope
- Child-safe learning assistant
- Teacher productivity tools
- Safety and supervision controls
- Retrieval over curated content
